================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "Python is not installed"
Solution: Install Python 3.8+ from https://www.python.org/
          Check "Add Python to PATH" during installation

Problem: "Virtual environment not found"
Solution: Run setup_venv.bat first

Problem: "Port 5000 already in use"
Solution: Edit .env file, change FLASK_PORT=5001

Problem: "Connection failed"
Solution: 1. Ping a265m001 to verify network
          2. Test credentials in SQL Server Management Studio
          3. Check .env file
